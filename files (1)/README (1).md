# Café Aurora ☕

Página web desenvolvida como trabalho prático da disciplina de Desenvolvimento Web, com foco na utilização de **HTML5** e **CSS3** puros para apresentação de uma empresa fictícia de torrefação artesanal de café.

---

## 📋 Sobre o projeto

O **Café Aurora** é uma cafeteria fictícia que serviu como tema para explorar conceitos de estruturação semântica em HTML e técnicas modernas de estilização em CSS. A página apresenta a história da empresa, seus produtos e informações de contato em uma interface visualmente atrativa e responsiva.

## 🚀 Como executar

1. Faça o download dos arquivos do projeto
2. Mantenha os dois arquivos (`index.html` e `style.css`) na mesma pasta
3. Abra o arquivo `index.html` em qualquer navegador moderno

Não é necessário instalar nada nem rodar servidor — basta abrir no navegador.

## 📁 Estrutura de arquivos

```
cafe-aurora/
├── index.html      # Estrutura da página
├── style.css       # Estilização completa
```

## 🛠️ Tecnologias utilizadas

- **HTML5** — estrutura semântica da página
- **CSS3** — estilização, animações e responsividade
- **Google Fonts** — fontes Playfair Display e Poppins
- **Font Awesome** — ícones decorativos

> Nenhum framework ou biblioteca de CSS foi utilizado, conforme exigência do trabalho.

## 🎨 Seções da página

- **Hero** — apresentação inicial com chamada principal
- **Sobre** — história da empresa e números em destaque
- **Produtos** — catálogo com três variedades de café
- **Contato** — informações da loja e formulário

## ✨ Inovações implementadas

Recursos de CSS aplicados além do conteúdo de sala de aula:

- **Variáveis CSS (Custom Properties)** — centralização de cores e valores no `:root`
- **Animações com `@keyframes`** — animações `surgir`, `flutuar`, `brilho` e `pulsar`
- **Texto com gradiente animado** — uso de `background-clip: text` para efeito de brilho no título
- **Glassmorphism** — efeito de vidro no cabeçalho usando `backdrop-filter: blur()`
- **`clip-path`** — recorte em formato poligonal nas imagens dos cards
- **Labels flutuantes** — animação dos rótulos do formulário com `:placeholder-shown`
- **Tipografia fluida** — uso de `clamp()` para escalonamento automático das fontes
- **Pseudo-elementos** — sublinhados animados no menu e detalhes decorativos
- **Responsividade** — layout adaptado para tablets e celulares com múltiplos breakpoints

## 📱 Responsividade

A página foi desenvolvida com abordagem mobile-friendly e possui breakpoints em:

- **900px** — adaptação para tablets
- **600px** — adaptação para celulares
